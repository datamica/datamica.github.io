# Datamica Website Project

Welcome to the Datamica website project! This repository contains the source code for the Datamica website, a data asset management consulting firm dedicated to helping SMEs and NGOs adopt a data-driven culture.

## Project Structure

The project is organized as follows:

```
datamica.github.io
├── index.html               # Main landing page
├── about.html               # About Us page
├── services.html            # Services page
├── processes.html           # Processes page
├── contact.html             # Contact Us page
├── components               # Reusable components
│   ├── header.html          # Header with navigation
│   ├── footer.html          # Footer with copyright
│   └── sections             # Sections for the landing page
│       ├── hero.html        # Hero section with slogan
│       ├── company-description.html # Company description placeholder
│       └── product-service.html     # Product and service offerings
├── assets                   # Static assets
│   ├── css
│   │   └── main.css         # Main CSS styles
│   └── js
│       └── main.js          # JavaScript for interactivity
├── data                    # Data files
│   └── services.json        # Structured data about services
├── package.json             # npm configuration file
└── README.md                # Project documentation
```

## Purpose

The purpose of this website is to provide information about Datamica's services and to promote a data-driven culture among small and medium enterprises (SMEs) and non-governmental organizations (NGOs). The name "Datamica" combines "data" and "amica," which means "friend" in Italian, reflecting our approachable and professional ethos.

## Getting Started

To get started with the project, clone the repository and open the `index.html` file in your web browser. You can also run a local server to view the website dynamically.

### Prerequisites

- A modern web browser
- (Optional) A local server setup for development

### Installation

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/datamica.github.io.git
   ```
2. Navigate to the project directory:
   ```
   cd datamica.github.io
   ```

3. Open `index.html` in your browser or set up a local server.
